The general course information

Welcome to Java.Functional programming course!

The course introduces elements of functional programming in Java 8+ including lambda expressions and method references, functional interfaces and function objects, stream API and lazy evaluation, currying, and monads. It gives you a practical approach to programming without too theoretical aspects of functional programming such as the Category Theory :)

The course syllabus consists of several modules revealing different aspects of functional programming in Java. Each module contains 1-2 theoretical lessons and a set of practice exercises. Some of these exercises are marked as HARD. If you don't have plenty of time, you can just skip them.

To solve our code challenges, you do not need to write the main method and I/O operations. They will be added to your solutions automatically. Such an approach allows you to focus on the meaning of the assigned tasks and reduce the spent time on completing exercises.

By the way, an improved version of the lessons is also available on Hyperskill which is a new project-based educational platform supported by Stepik and JetBrains.

Note 1. For some historical reasons, all the theories and practice lessons here are separated, and each practice lesson consists of only one step unlike many other Stepik courses. There is no easy way to change this without losing the progress of our learners. We wouldn't like to do this, so please be understanding.

Note 2. There are some mistakes in English in the course because 3 years ago the author didn't know English well enough to avoid them. But we are going to fix them soon. Anyway, the mistakes shouldn't reduce your understanding of the lessons.

If you have any questions or suggestions, write them in comments. Happy learning!
